 # ImpactBridge

ImpactBridge is a fullstack donation platform where users can create campaigns, donate to causes, track their impact, and use AI-powered analysis to improve campaign effectiveness.

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: Go
- Database: PostgreSQL
- Payments: Stripe test mode
- AI: OpenAI API
- Deployment: Vercel + Render

## Core Roles

- Admin
- Campaign Creator
- Donor

## MVP Features

- User authentication
- Campaign creation and publishing
- One-time donations
- Recurring donations
- Pledges
- Donor dashboard
- Creator dashboard
- Admin dashboard
- Hall of Fame leaderboard
- AI campaign analysis
